\section{Data and Repository}

	All the data used can be found in Morgana at the folder \newline \filename{/morgana2/snert/PhotonicLanternInvestigationCarmona2024/Data}:
	
	\begin{itemize}
		\item The folder \filename{AmpPhaseReconstruction/} contains the data used for SLM Reconstruction
		\item The folder \filename{PSFReconstruction/} contains the data for PSFReconstruction and Photonic Lantern Information Determination
		\item The folders \filename{NMIAnalysisDatasets} and \filename{PSFReconstructionDatasets} contain the data for Clustering comparison and Adjusted Mutual Information Analysis.
		\item The folder \filename{LastDance} contains the data for Adjusted Mutual Information Analysis with an arbitrary matrix.
	\end{itemize}
	
	All the code can be found in the github repository\href{https://github.com/Dacarpe03/PLImageReconstruction}{PLImageReconstruction}.
	
	All the plots used in the \filename{appendix} are in the repository folder \href{https://github.com/Dacarpe03/PLImageReconstruction/tree/main/LabNotebook/Experiments/ExpImages}{LabNotebook/Experiments/ExpImages}
